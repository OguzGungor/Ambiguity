# Ambiguity Server Side
It is a branch for server side of IQ Puzzler Pro Versus Mode.

## Installation
Start the server:
run `` mosquitto server ``
run `` start.bat ``

#### Protocols
* MQTT
* UDP

#### Library
* Eclipse Paho MQTT
* JSON

#### Features
* Loby (up to 4 client)
* Loby Chat
* Instant update after clients' request
* Own game management
